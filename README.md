📦 セットアップ方法
# リポジトリをクローン
git clone https://github.com/your-username/your-repository.git

# ディレクトリへ移動
cd your-repository

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

.env ファイルを作成し、以下を設定してください。

SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_anon_key
